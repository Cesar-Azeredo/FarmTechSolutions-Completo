|---------|----------------|-------------|--------------|

- **pH Mínimo:** 5.5
- **pH Ideal:** 6.5
- **pH Máximo:** 7.5

- **Mínima (irrigar):** < 40%
- **Ideal:** 60%
- **Máxima (encharcado):** > 80%
